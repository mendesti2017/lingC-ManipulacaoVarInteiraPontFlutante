# lingC-ManipulacaoVarInteiraPontFlutante
Demonstrar como manipular variáveis inteiras e de ponto flutuante, além de aprender sobre a conversão entre tipos de dados. 

Conversão implicita - ocorre automaticamente quando você mistura diferentes tipos de dados em uma expressão. O Computador converte os tipos de dados para realizar a operação de acordo com o que o programador escreveu.

Conversão explícita (casting) - ocorrer quando é usado os operadores de casting para forçar a conversão de um tipo de dado para outro.
